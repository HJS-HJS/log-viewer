### How to Use
- Installation
  - The installation method is the same for both Windows and Ubuntu.
    ``` bash
    pip install pyinstaller
    pyinstaller --onefile --windowed log_viewer.p
    ``` 
- Run
  - Window
    ``` bash
    .\dist\log_viewer.exe
    ``` 
  - Ubuntu
    ``` bash
    ./log_viewer
    ```
